Jeux de carte 

### Installation

```sh
$ git clone https://github.com/speeeeeed2006/CardsGame.git
$ cd CardsGame
$ composer install
```

### Utilisez le serverless Symfony

```sh
$ php bin/console server:start
```
 [http://127.0.0.1:8000](http://127.0.0.1:8000)

### Unit testing

```sh
$ php vendor/bin/phpunit tests/
```
